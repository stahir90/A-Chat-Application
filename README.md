# A-Chat-Application

chat application based on React.js, Typescript, Hooks, Material-UI, Socket.io, Node.js, Express


# Run clientApp
1) Node version 10+
2) cd clientApp
3) yarn 
4) yarn start


# Run server
1) Node version 10+
2) cd server
3) npm i 
4) npm run start
